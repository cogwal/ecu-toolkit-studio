# Folder that contains all native libraries that the toolkit studio depends on.

## Toolkit Libraries
1. Download the proprietary toolkit library version 1.4.0
2. Copy the content of the folder `Binaries\Library` found in the zip file into the folder `toolkit`
3. Copy the folder `Include` found in the zip file into the folder `toolkit`

## PCAN Basic Drivers
1. Download the [PCAN-Basic](https://www.peak-system.com/PCAN-Basic.239.0.html?&L=1) API form the Peak-System website.
2. Copy the content of the downloaded ZIP file into the folder `PCAN-Basic`
